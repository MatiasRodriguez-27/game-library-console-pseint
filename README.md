# 🎮 Game Library (C & PSeInt)

Collection of interactive console-based games developed using C and PSeInt, focused on practicing programming fundamentals and logical thinking.

## 🚀 Features

* Multiple game implementations using decision-making logic
* Use of control structures (conditionals and loops)
* Input validation and user interaction handling
* Modular and organized code structure
* Reusable logic across different programs

## 🛠️ Technologies

* PSeInt

## 🎯 Purpose

This project was created to strengthen problem-solving skills and core programming concepts such as control flow, algorithms, and structured programming.

## ▶️ How to Run

1. Open the project in PSeInt or a C compiler
2. Select the desired program/game
3. Run the file and follow the console instructions
